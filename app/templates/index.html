About This Event

Learn how to use Python for data analysis and visualization in this full-day workshop. This workshop is perfect for beginners who want to get started with data science using Python.

Topics covered include:

  • Introduction to Python for data analysis
  • Data manipulation with pandas
  • Numerical computing with numpy
  • Data visualization with matplotlib and seaborn
  • Introduction to machine learning with scikit-learn

Participants should bring their own laptops with Python installed. We will provide instructions for setting up your environment before the workshop.
